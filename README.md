#  Nutrition Tracker Application

A full-stack web application that helps users log daily meals and analyze nutritional intake such as calories, protein, carbohydrates, and fats using AI-powered analysis.

---

##  Features

- User authentication (Sign up / Login)
- Maintenance calorie calculator
- Daily meal logging
- AI-based nutrition analysis (calories & macros)
- View nutrition summary day-wise
- Edit or delete meal logs
- Responsive and user-friendly UI

---

##  Tech Stack

### Frontend
- ReactJS
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI Integration
- Gemini 1.5 API

### Authentication & Security
- JWT (JSON Web Tokens)
- Bcrypt (password hashing)

---

##  How It Works

1. User registers or logs in to the application  
2. User calculates maintenance calories based on personal details  
3. Meals are entered with food information  
4. Meal data is sent to the AI model  
5. AI returns calorie and nutrition breakdown  
6. Results are stored and displayed on a daily basis
